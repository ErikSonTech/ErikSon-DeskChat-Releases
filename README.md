# ErikSon Message — Releases

Release feed and binaries for [ErikSon Message for WhatsApp](https://github.com/ErikSonTech/ErikSon-Message-Releases/releases).

## Download

Download the latest DMG from [Releases](https://github.com/ErikSonTech/ErikSon-Message-Releases/releases).

### First install
1. Open the DMG and drag the app to Applications
2. Right-click the app → Open (required for first launch, since the app is not notarized)
3. After that, updates are automatic via Sparkle
